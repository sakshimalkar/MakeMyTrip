# ✈️ PackMate - Travel Planning Web App

PackMate is a modern travel planning web application that helps users organize trips, manage packing lists, and track progress efficiently.

---

## 🌐 Live Demo

👉 [View Live Demo](https://sakshimalkar.github.io/MakeMyTrip/)

---

## 🚀 Features

- 🏠 Dashboard with trip overview
- 📋 Master Template for packing essentials
- ✈️ Create and manage trips
- ✅ Track packing progress
- 📊 Visual indicators for completion status
- 🎯 Clean and responsive UI

---

## 🛠️ Tech Stack

- Frontend: React (Vite)
- Styling: CSS
- State Management: React Hooks
- Build Tool: Vite

---

git clone https://github.com/sakshimalkar/MakeMyTrip.git
