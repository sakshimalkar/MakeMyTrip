## ✨ Features

| Feature | Description |
|---------|-------------|
| 🏠 **Dashboard** | Clean trip overview with all your travels at a glance |
| 📋 **Master Template** | Pre-built packing essentials so you never forget anything |
| ✈️ **Trip Manager** | Create, edit & manage multiple trips easily |
| ✅ **Packing Tracker** | Check off items as you pack in real time |
| 📊 **Progress Indicators** | Visual completion status for each trip |
| 📱 **Responsive UI** | Works perfectly on mobile, tablet & desktop |

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/sakshimalkar/MakeMyTrip.git

# Navigate into the project
cd MakeMyTrip

# Install dependencies
npm install

# Start development server
npm run dev
```
